Premium Animations — Final

Génériques, sans texte, utilisables partout.

1. Convergence — App launch, page load, splash screen

ENTRANCE
BindX
In Silico Drug Discovery Platform
Les 3 cercles volent depuis 3 directions avec rotation + traînées. Convergence élastique, puis respiration idle infinie. Utilisable : splash, hero, page transition.

2. Loading — Calcul en cours, traitement, import

PROCESSING
Les 3 cerceaux tournent autour du point central avec accélération/décélération sinusoïdale. Le point se déplace sous les forces centrifuges proportionnelles à la masse de chaque cerceau. Utilisable : calcul, import, scoring, n'importe quel run.

3. Flux — Screening, filtrage, pipeline actif

STREAMING
Des particules de tailles variées coulent de gauche à droite, rétrécissent et disparaissent en traversant chaque cercle. Le logo est le funnel. Utilisable : screening, import library, data flow.

4. Succès — Tâche terminée, validation, résultat

SUCCESS
Les cercles convergent serrés → burst mint radial + 10 sparkles multicolores → point vert avec ✓ animé. Utilisable : run terminé, phase validée, export réussi.

5. Erreur — Échec, timeout, crash

ERROR
Même structure que le succès mais tout rouge. Les cercles convergent → burst rouge + sparkles rouges → point rouge avec ✕ animé. Symétrique au succès. Utilisable : crash, timeout, erreur réseau.

6. Transition — Changement de phase, avancement, shift

TRANSITION
Le 1er cercle sort par la gauche, revient avec la couleur du suivant. Tout le spectre avance (mint→cyan→blue → cyan→blue→violet). Utilisable : changement de phase, avancement, navigation entre étapes.

7. Exploded — Hover, inspection, détail

INTERACTIVE
Screen Filter Enhance
↑ Survolez le logo
Les cercles s'écartent avec spring, les labels apparaissent à l'extérieur (gauche, haut, droite) avec lignes pointillées. Utilisable : hover nav, tooltip, onboarding, documentation.

8. Magnétique — Réaction au curseur en temps réel

REACTIVE
↑ Déplacez votre curseur
Les cercles réagissent à la position du curseur en temps réel. Le grand cercle suit lentement, le moyen plus vite, le petit est aimanté. Parallax inversé. Utilisable : hero, landing page, background décoratif.
